# Olympic Performance Analytics

## Objective
Analyze Olympic performance trends using historical Olympic data.

## Features
- KPI Dashboard
- Year-wise filtering
- Top Countries by Medal Count
- Medal Trends Over Years
- Top Athletes Analysis
- Top Sports Analysis
- Gender Participation Analysis

## Technologies Used
- Python
- Pandas
- Plotly
- Streamlit

## Run Locally

streamlit run app.py
