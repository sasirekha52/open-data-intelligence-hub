EXTREME WEATHER INTELLIGENCE PLATFORM
======================================

REQUIREMENTS:
- Python 3.8+
- polars library
- matplotlib (for visualizations)

INSTALLATION:
pip install polars matplotlib

RUNNING:
1. Place all CSV files in same directory as notebook
2. Open extreme_weather_platform.ipynb in Jupyter
3. Run all cells (Kernel → Restart & Run All)

OUTPUT:
- extreme_weather_processed.csv (processed data with event labels)
- Console output showing event summary and most severe events
