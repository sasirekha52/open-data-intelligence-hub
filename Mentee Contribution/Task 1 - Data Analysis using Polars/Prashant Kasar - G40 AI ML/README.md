# 📊 Literacy Trend Intelligence Engine

## 🚀 Project Overview
This project analyzes literacy trends using data analysis and visualization techniques. It provides insights into literacy growth, country comparisons, forecasting, and dashboard-based analytics.

---

## 📁 Project Structure

- app.py → Main application
- pages/ → Dashboard, Trend Analysis, Country Comparison, Forecasting, Reports
- models/ → Data processing and logic
- notebooks/ → Jupyter analysis
- screenshots/ → Output visuals
- docs/ → Project report
- literacy_data.csv → Dataset

---

## ⚙️ Features

- 📊 Interactive Dashboard
- 🌍 Country-wise Literacy Comparison
- 📈 Trend Analysis
- 🔮 AI Forecasting
- 📑 Reports Generation
