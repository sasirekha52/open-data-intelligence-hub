# Customer Segmentation Project

This workspace contains a complete customer segmentation mini-project that uses synthetic customer and transaction data to demonstrate:

- RFM-style customer behavior features
- K-Means clustering for segmentation
- Regression for spending prediction
- Classification for purchase likelihood
- Hyperparameter tuning and business-focused reporting

## Project Structure

- data/ - generated CSV datasets
- notebooks/ - notebook workspace
- src/ - reusable Python modules
- reports/ - generated segment reports and insights

## Run the Project

From the workspace root, run:

```bash
"d:/Customer Segmentation/.venv/Scripts/python.exe" src/run_pipeline.py
```

## Deliverables

- customer_segments.csv
