# Model Evaluation Report

## Customer Churn Prediction using Random Forest

---

## 1. Project Overview

The objective of this project is to predict whether a telecom customer is likely to churn (leave the service) based on customer demographics, account information, and subscribed services. A reusable Machine Learning Pipeline was built using Scikit-learn to automate data preprocessing and model training.

---

## 2. Model Used

**Algorithm:** Random Forest Classifier

**Reason for Selection:**

* Handles both numerical and categorical features effectively.
* Reduces overfitting by combining multiple decision trees.
* Provides good performance on classification problems.
* Produces reliable and stable predictions.

---

## 3. Dataset Information

**Dataset Name:** Telco Customer Churn Dataset

| Attribute       | Value |
| --------------- | ----- |
| Total Records   | 7043  |
| Total Features  | 21    |
| Target Variable | Churn |

---

## 4. Data Preprocessing

The following preprocessing steps were performed before training the model:

* Removed the **customerID** column because it is a unique identifier and does not contribute to prediction.
* Converted the **TotalCharges** column from object type to numeric.
* Replaced blank values in **TotalCharges** with **NaN**.
* Filled missing numerical values using **Median Imputation**.
* Filled missing categorical values using **Most Frequent Imputation**.
* Applied **One-Hot Encoding** to categorical features.
* Applied **Standard Scaling** to numerical features.
* Combined preprocessing steps using **ColumnTransformer**.
* Created a reusable Machine Learning Pipeline using **Pipeline**.

---

## 5. Train-Test Split

| Parameter        | Value |
| ---------------- | ----- |
| Training Data    | 80%   |
| Testing Data     | 20%   |
| Random State     | 42    |
| Stratified Split | Yes   |

---

## 6. Model Training

The Random Forest Classifier was trained using the training dataset through the Scikit-learn Pipeline. During training, all preprocessing steps including missing value handling, feature encoding, and feature scaling were automatically applied before fitting the model.

---

## 7. Model Performance

### Accuracy Score

**Accuracy:** **77.79%**

The model correctly classified approximately **77.79%** of the customers in the test dataset.

## 10. Conclusion

A complete Machine Learning Pipeline was successfully developed for Customer Churn Prediction using the Random Forest Classifier. The pipeline automatically performs data preprocessing, including handling missing values, encoding categorical variables, scaling numerical features, and generating predictions.

The trained model achieved an accuracy of **77.79%**, demonstrating satisfactory performance for the customer churn classification task. Since preprocessing and model training are integrated into a single reusable pipeline, the model can easily be used for predicting churn on new customer data.

Overall, the project demonstrates the practical implementation of an end-to-end Machine Learning workflow using Scikit-learn's Pipeline and ColumnTransformer.
