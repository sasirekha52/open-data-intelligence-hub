# 📌 Project Overview

The **SmartShop Recommendation System** is an end-to-end Machine Learning project that analyzes customer shopping behavior and provides business insights using multiple machine learning algorithms.

The project demonstrates the complete Data Science lifecycle:

- Data Collection
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Machine Learning
- Customer Segmentation
- Streamlit Dashboard
- Business Insights

---

# 🎯 Project Objectives

- Predict Product Ratings
- Predict Purchase Status
- Segment Customers
- Compare Machine Learning Models
- Generate Actionable Business Insights

---

# 🧠 Machine Learning Algorithms

### 📉 Ridge Regression

Predicts Product Ratings.

---

### 🎯 Logistic Regression

Predicts whether a customer will purchase a product.

---

### 👥 K-Means Clustering

Segments customers into meaningful business groups.

# 🚀 How to Run

## Clone Repository

```bash
git clone https://github.com/yourusername/SmartShop_Recommendation_System.git
```

---

## Open Project

```bash
cd SmartShop_Recommendation_System
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv
```

---

Activate

Command Prompt

```bash
venv\Scripts\activate
```

PowerShell

```bash
.\venv\Scripts\Activate.ps1
```

---

## Install Libraries

```bash
pip install -r requirements.txt
```

---

## Run Streamlit

```bash
cd app

streamlit run streamlit_app.py
```
